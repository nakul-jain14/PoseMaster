About My PoseNet Project

The PoseMaster is the project that employs machine learning to detect the pose of a person in real-time using a webcam, utilizing the PoseNet library, which is part of the ML5.js library. PoseNet is a pre-trained model that can estimate the pose of a person within an image or video stream. Notably, the model has been trained on a dataset that includes poses forming the letters Y, A, M, and C. As a result, when the corresponding poses are struck in front of the webcam, the respective letter is displayed on the screen. This innovative feature allows users to interact with the model in a fun and engaging way.

How it Works

Upon clicking the "Get Started" button, the application initiates the webcam, enabling real-time video processing. The PoseNet model then detects the user's pose, represented by key points, and visualized through a stick figure overlay on the video stream. The model is specifically trained to recognize four distinct alphabet poses: Y, A, M, and C. When the user assumes one of these poses, the corresponding letter is displayed on the screen, creating a dynamic and interactive user experience. This project showcases the power of machine learning and real-time video processing in detecting and visualizing human pose.
